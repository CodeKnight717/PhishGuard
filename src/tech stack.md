# Technology Stack

## Backend
- **Language**: Python 3.12+
- **Framework**: FastAPI (High performance, easy to use, async support)
- **Server**: Uvicorn
- **Database**: PostgreSQL
- **ORM**: SQLAlchemy (Async)
- **Validation**: Pydantic
- **Testing**: Pytest

## Frontend (Browser Extension)
- **Framework**: React
- **Build Tool**: Vite
- **Extension Tooling**: CRXJS (Vite plugin for Chrome Extensions)
- **Styling**: TailwindCSS
- **State Management**: Zustand or React Context
- **Icons**: Lucide React

## AI / Detection Engine
- **Libraries**: Scikit-learn, TensorFlow (if needed for deep learning)
- **APIs**: Google Safe Browsing API, PhishTank API

## DevOps
- **Containerization**: Docker
- **CI/CD**: GitHub Actions
